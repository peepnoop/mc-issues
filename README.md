# mc-issues

**Submit a new issue for any features or bugs for a minecraft server.**

# Contributing

To add a new issue click on the issues tab and use one of the templates for your request. If neither template fits your need then create a generic issue.
**Bug Report:** This is for any unintended issues with the server
**Feature Request:** For adding new features to the server such as datapacks,modpacks,etc. This can also be used for refining or removing currently installed features.
